This is a data file for the representatives of the 24th congress of the Grand National Assembly of Turkey.
Sercan Pekel was helpful in gathering these data.
The contents of the data file are publicly available and can be used for purposes of statistical analysis.
The variables are: 1)Name, 2)Birthdate (year), 3)Sex (coded as M for male and W for female), 4)Profession, and 5)Lang.
The variable Lang is for the (self-declared) number of languages spoken by the deputies.
